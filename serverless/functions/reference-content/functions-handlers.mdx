## Node, Python, PHP

For Node, Python and PHP functions, the handler must be written in the `<folder>/<file>.<function_name>` format.

- If the `handle` function is defined in a file named `handler.py`: the handler name will be `handler.handle`.

- If the `handle` function is defined in a folder (for example, `handlers/handler.py`): the handler will be `handlers/handler.handle`.

## Go and Rust

For Go and Rust, the handler must be written in the `<folder>/<function_name>` format: 

- If the `Handle` function is defined in a `handler.go` file in the root folder: the handler name will be `Handle`.

- If the `Handle`function is defined in a `handler.go` file (for example, `handlers/handler.go`): the handler will be `handlers/Handle`

<Message type="note">
  In Go, the method name must be capitalized.
</Message>


the function needs to be exported (in Go capitalize the method name, in Rust define the method as "pub").

    the `Handle` function is in a file named `handler.go`: the handler will be `Handle`
    the `Handle` function is defined in a folder (like `handlers/handle.go`): the handler will be `handlers/Handle`
